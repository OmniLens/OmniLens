## Date Selection

The dashboard allows you to view workflow runs for any date. Use the date controls in the top-right corner:

- **Today button**: Quickly jump to today's date
- **Calendar picker**: Select any date to view historical workflow runs
- **Refresh button**: Reload the page to fetch the latest data

## Daily Metrics Overview

The dashboard displays four metric cards showing aggregated statistics for the selected date.

### Pass/Fail Rate

A pie chart showing the success rate of workflow runs:

- **Percentage**: Displayed in the center of the chart
- **Passed runs**: Count of successful workflow runs
- **Failed runs**: Count of failed workflow runs

### Overview

General workflow statistics:

- **Workflows**: Total number of workflows configured in the repository
- **Completed**: Total number of completed workflow runs
- **Didn't run**: Number of workflows that didn't run on the selected date
- **Total Runtime**: Cumulative runtime of all workflow runs (formatted as hours, minutes, and seconds)

### Workflow Health

Health status breakdown comparing today's runs with yesterday's:

- **Consistent**: Workflows that are stable and passing
- **Improved**: Workflows that got better compared to yesterday
- **Regressed**: Workflows that got worse compared to yesterday
- **Still failing**: Workflows that continue to fail

### Runs by Hour

A bar chart showing when workflows are most active throughout the day:

- **Hourly distribution**: Stacked bars showing passed and failed runs per hour
- **Activity patterns**: Identify peak workflow activity times

## Workflow Cards

## Workflow  Card Overview

Each workflow card provides a compact overview of the workflow's activity.

![Workflow Card](/workflow-card.png)

### Metrics

The repository card displays the following metrics:

- **Status badge**: Visual indicator showing Pass, Fail, Running, or Idle
- **Run count badge**: Shows when a workflow runs multiple times
- **Health status indicator**: Compares today's performance with yesterday (Consistent, Improved, Regressed, or Still Failing)
- **Duration**: The total runtime of the workflow execution

### Actions

Each repository card includes interactive elements:

- **View button**: Opens the workflow run on GitHub Actions
- **Run count badge**: Click to view all runs
