## Features

### Repository Management

Add, validate, and manage repositories within your dashboard to monitor workflow health.

### Workflow History

Explore workflow runs across specific dates using a clear, calendar-based history view.

### Metrics & State

Track workflow health using success rates, run counts, runtimes, and stability trends over time.
